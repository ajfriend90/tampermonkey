# Asana Rack Helper

Tampermonkey userscript for creating and populating Asana rack tasks from asset IDs.

## Features
- Validates asset IDs
- Pulls data from Boost and RACKS
- Auto-fills Asana custom fields
- Clear warnings and failure reporting

## Installation
1. Install Tampermonkey
2. Open:
   https://raw.githubusercontent.com/ajfriend90/tampermonkey/main/asana-rack-helper/asana-rack-helper.js
3. Click **Install**

## Updating
Updates automatically via Tampermonkey when changes are merged to `main`.

## Development
All changes should be made on the `develop` branch and merged into `main` when stable.
